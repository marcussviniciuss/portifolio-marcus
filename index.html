Hotel Management System

Clique aqui!
Este projeto é um Sistema de Gerenciamento Hoteleiro desenvolvido para atender às necessidades específicas de hotéis, oferecendo controle eficiente sobre quartos alugados, reservas e operações do pessoal. Suas principais funcionalidades incluem o cadastro de quartos, controle em tempo real do status dos quartos e gerenciamento da rotina dos funcionários, com diferentes níveis de acesso para garantir eficiência operacional. Destaca-se também a segurança avançada, com senhas criptografadas e criação segura de novos usuários. O sistema utiliza tecnologias como banco de dados SQLite para armazenamento confiável, backend em Python e Flask para robustez e flexibilidade, e frontend com HTML e CSS para uma interface atraente e amigável.
